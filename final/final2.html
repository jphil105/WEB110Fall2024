Heats of reaction

Enthalpy changes of a reaction can be determined by stoichiometric ratios.

These stoichiometric ratios can be useful for moving from one conversion factor to another.

  1. Start with the given value
  2. Use your given value to start dimensional analysis
  3. Use your descendent values to continue the dimensional analysis
  4. Use any given enthalpy value to end the dimensional analysis
  5. End your answer with the correct unit

Heats of formation

Heats of formation are standardized so we can use chemical equations to derive those standards.

The formula is that you take the sum of all standard enthalpies of formation of all products minus the enthalpies of the reactants.

ΔH°reaction=∑nΔH°f(products)‐∑nΔH°f(reactants)

Hess's law

Hess's law states that the sum of the enthalpies of similar reactions are equal to the enthalpy of an unknown reaction.

We can do this by using a similar method in algebra, and that method involves eliminating unwanted values.

Once we eliminate all unwanted reactants/products, we can get the answer to the unknown.

If you don't want to use Hess's law, then you could use stoichiometric ratios instead
